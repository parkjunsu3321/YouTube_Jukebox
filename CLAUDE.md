# 프로젝트 개요
해당 프로젝트는 유튜브 영상을 틀어주는 주크박스 사이트가 주제인 프로젝트야.

## 기술 스택
백엔드 : FastAPI(웹 소켓 사용)
프론트 엔드 : React(웹 소켓 사용)
DB : SQLite

## 기능
0. 기능 구분
기능은 크게 호스트, 클라이언트로 나눌꺼야. 호스트는 .../host로 입장하도록 할꺼야. 클라이언트는 / 주소로 들어가게 할꺼야.

1. 호스트 기능정리
1-1. 노래 재생
호스트는 클라이언트가 재생 버튼 클릭 시에 바로 노래가 재생되게하는 기능

1-2. 플레이 리스트 가져오기기
호스트는 한 곡이 끝나면 다음 곡 시작되기 전에 리스트를 가져오는 기능

1-3. 호스트 재생 버튼
호스트에서 재생을 할 수 있도록 하는 기능능

2. 클라이언트 기능
2-1. 노래 추가
노래를 유튜브 url로 추가하거나, 노래를 검색해서 할 수 있게해서 검색을 해서 추가할 수 있게하는 기능

2-2. 노래 재생
노래 재생 버튼을 누르면 소켓을 통하여 호스트에서 노래가 재생될 수 있게 하는 기능

## 작동 흐름
1. 호스트 on -> 2. 클라이언트 접속 -> 3. 클라이언트 노래 추가 -> 4. 클라이언트 노래 재생 클릭 -> 5. 호스트 노래 재생
위 흐름이 가장 기본적인 흐름이야.
주의 사항
1) 4에서 5로 넘어갈때, 다른 클라이언트가 노래를 추가해도 아무 문제 없이 하게해줘
2) 호스트가 재생 상태일때, 클라이언트가 재생버튼을 누르면 "호스트가 이미 재생상태입니다."라는알람이 뜨게해줘
